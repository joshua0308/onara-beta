## Getting started
`npm run dev` - start server with nodemon (port 3000)

## Endpoints
- `/login` - login endpoint
- `/game` - game endpoint
- `/status` - status endpoint

## How to deploy
`git push heroku HEAD:master` - push your HEAD to master in heroku's remote
